# Planetas

## Descripción
Resolución de un problema de Dijkstra modificado de Codeforces en C++.

## Idea
Se utiliza una variante de Dijkstra teniendo en cuenta tiempos en los que no se puede salir de ciertos nodos.

## Tecnologías
- C++

## Complejidad
O((n + m) log n)

## Cómo ejecutar
g++ -o viaje.exe viaje.cpp  
.\viaje.exe

## Problema: https://codeforces.com/contest/230/problem/D
